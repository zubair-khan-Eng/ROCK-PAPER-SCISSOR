# Rock Paper Scissor Game ✂️

A fun Rock Paper Scissor game built with HTML, CSS, and JavaScript.

## 🎮 How to Play
- Choose Rock, Paper, or Scissor
- Computer randomly picks its choice
- Winner is decided based on classic rules

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
